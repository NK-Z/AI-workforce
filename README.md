# AI Workforce — Virtual Office

A modern, interactive web application for managing AI agents as employees in a virtual office environment. Hire agents, assign tasks, chat with them, and monitor performance from a beautiful dashboard.

## Features

- **Dashboard** — Real-time stats, activity feed, and Kanban task board
- **Office Floor Plan** — Interactive 2D office layout with rooms, desks, and agent positions
- **Agent Management** — Hire, configure, and monitor AI agents with different roles and skills
- **Task Management** — Create, assign, and track tasks across a Kanban board
- **Chat** — Simulated conversations with agents based on their roles
- **Analytics** — Performance metrics, charts, and insights
- **PWA Support** — Installable as a progressive web app with offline caching
- **Dark/Light Mode** — Toggle between themes

## Getting Started

No build tools required. Simply open `index.html` in a modern browser, or visit the live deployment.

## Live Demo

Deployed via GitHub Pages at: `https://nk-z.github.io/AI-workforce/`

## Tech Stack

- Pure HTML5 / CSS3 / JavaScript (ES6+)
- No frameworks or dependencies
- localStorage for persistence
- Service Worker for offline support
- GitHub Actions for CI/CD deployment
